# CAELab UI 重设计规范

## 一、设计目标

将 CAELab 从"MVP原型"提升为"可发布的专业CAE软件界面"。

参考风格：**印象笔记的清爽感 + GoodNotes的简洁工具感 + COMSOL的工程专业感**

---

## 二、整体设计语言

### 2.1 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 主色 | `#2563EB` | 蓝色，工具激活态、按钮主色 |
| 主色浅 | `#3B82F6` | hover态、图标 |
| 次色 | `#10B981` | 成功状态、求解完成 |
| 警告色 | `#F59E0B` | 警告提示 |
| 危险色 | `#EF4444` | 错误、应力超限 |
| 背景色 | `#FAFBFC` | 主背景（参考印象笔记白） |
| 卡片色 | `#FFFFFF` | 面板、卡片 |
| 边框色 | `#E5E7EB` | 分割线、输入框边框 |
| 侧边栏色 | `#F3F4F6` | 左侧导航背景 |
| 深文字 | `#1F2937` | 标题、重要文字 |
| 中文字 | `#4B5563` | 正文 |
| 浅文字 | `#9CA3AF` | 辅助说明、占位符 |

**CAE专业色板（仿真结果可视化）：**
- 低温区：`#3B82F6`（蓝）
- 中温区：`#22C55E`（绿）
- 高温区：`#EF4444`（红）
- 应力云图：`#6366F1` → `#EC4899`（紫粉渐变）

## 三、布局结构

### 3.1 整体框架（五区布局）

```
┌─────────────────────────────────────────────────────────┐
│  顶部标题栏（48px）                                       │
├────────┬────────────────────────────────┬────────────────┤
│        │                                │                │
│  左侧  │        主工作区                 │    右侧面板    │
│ 导航栏  │   （3D视图/表单/图表）           │  （属性/参数）  │
│（64px）│                                 │   （280px）    │
│        │                                 │                │
├────────┴────────────────────────────────┴────────────────┤
│  底部状态栏（32px）                                       │
└─────────────────────────────────────────────────────────┘
```

### 3.2 顶部标题栏（48px）

```
[Logo+产品名]    [项目名称 / 当前模型]    [用户菜单] [设置] [帮助]
```

- 左侧：CAELab Logo（24px） + 产品名
- 中间：当前打开的项目/文件名，可点击切换
- 右侧：通知铃铛 → 用户头像下拉菜单

### 3.3 左侧导航栏（64px宽，图标模式）

| 图标 | 模块 | 说明 |
|------|------|------|
| 🏠 | 首页 | 项目总览 |
| 📐 | 建模 | 几何建模 |
| ⚙️ | 仿真 | 仿真设置与求解 |
| 🔋 | 疲劳 | 疲劳分析 |
| 📊 | 后处理 | 结果可视化 |
| 📝 | 笔记 | 仿真记录笔记 |
| 💻 | 代码 | 脚本/自定义 |
| ⚡ | 自动化 | 仿真流程自动化 |

**导航激活态：** 左侧蓝色指示条（3px）+ 图标颜色变主蓝

### 3.4 主工作区

根据模块不同切换：

**首页：** 项目卡片网格（最近项目）+ 快捷入口
**建模：** 左侧工具栏 + 中央3D视图 + 右侧属性面板
**仿真：** 上方标签页（静态/瞬态/疲劳…）+ 中央参数表单 + 右上面板
**后处理：** 全屏3D结果视图 + 底部曲线图（可拖拽分割）

### 3.5 右侧面板（280px，可折叠）

默认显示当前模块的关键参数，类似 COMSOL：
- 材料属性
- 边界条件
- 网格设置
- 求解器控制

### 3.6 底部状态栏（32px）

```
[求解器状态]  [进度条]  [当前时间步]  [内存占用]  [GPU状态]  [版本号]
```

## 四、核心模块详细设计

### 4.1 首页（HomeView）

布局：两栏
- 左：最近项目列表（卡片式，每卡显示名称+修改时间+状态标签）
- 右：快捷入口（新建仿真/打开模型/帮助文档）

**项目卡片设计：**
```
┌─────────────────────────────┐
│  [模型缩略图 140x90]          │
├─────────────────────────────┤
│  支架静态分析                 │  ← 项目名
│  修改于 2小时前   ✓ 已完成    │  ← 时间 + 状态
│  [打开]  [删除]              │  ← 操作
└─────────────────────────────┘
```

### 4.2 仿真模块（SimulationView）

采用**标签页式**设计，不同分析类型切换标签：

```
[静态分析] [瞬态分析] [模态分析] [疲劳分析] [热耦合] [+更多]
```

每个标签页内：
- 左：参数输入表单（分类折叠面板：几何/材料/边界/荷载/求解）
- 右：3D模型预览（只读，随参数联动更新）
- 底部：求解控制条

**材料输入面板参考 COMSOL：**
```
▼ 材料属性
   材料库 [▼ 选择]  [从项目材料库选择]
   名称：铝合金 6061
   弹性模量：68.9 GPa  [输入] [单位]
   泊松比：0.33        [输入]
   密度：2.70 g/cm³    [输入]
   屈服强度：276 MPa   [输入]
```

### 4.3 后处理模块（结果可视化）

核心设计：**全屏3D + 底部图表浮动面板**

```
┌────────────────────────────────────────────────────┐
│ [返回]  支架静态分析 - 等效应力   [+ 导出] [+ 截图] │
├────────────────────────────────────────────────────┤
│                                                    │
│            3D 结果云图（全屏）                      │
│                                                    │
│   [云图图例条]                                      │
│   Min: 0    Max: 276 MPa                           │
│   [蓝━━━渐变━━━红]                                  │
│                                                    │
├────────────────────────────────────────────────────┤
│ [变形显示][剖切面][极值点][动画] │  选择: [▼应力类型]│
├────────────────────────────────────────────────────┤
│  📈 响应曲线（可拖拽改变高度）                        │
│  X轴: [时间/位置]  Y轴: [应力/位移]                  │
└────────────────────────────────────────────────────┘
```

### 4.4 笔记模块（NotesView）

参考印象笔记+GoodNotes：
- 左侧：笔记列表（搜索栏 + 标签过滤 + 笔记卡片列表）
- 右侧：富文本编辑器（支持公式、图片、表格）

**笔记卡片：**
```
┌─────────────────────────────────┐
│ 支架仿真-20260401               │ ← 标题
│ ...仿真参数记录...               │ ← 摘要（2行截断）
│ 📅 2026-04-01  🔖 CAE  📊 3张图  │ ← 元信息
└─────────────────────────────────┘
```

## 五、交互规范

### 5.1 通用交互

| 操作 | 效果 |
|------|------|
| 点击按钮 | scale(0.98) 50ms + 颜色变深 |
| 卡片悬停 | 阴影加深 + 边框微蓝 |
| 面板展开/折叠 | 200ms ease-out 高度过渡 |
| 标签切换 | 下划线滑动指示器 200ms |
| 加载状态 | 骨架屏（shimmer动画） |
| 求解进度 | 进度条 + 百分比 + ETA |

### 5.2 3D视图交互

- 旋转：鼠标左键拖拽
- 平移：鼠标中键 / Shift+左键
- 缩放：滚轮
- 拾取：单击选择（高亮+属性面板更新）
- 右键菜单：显示快捷操作

### 5.3 表单交互

- 实时验证：失焦时校验 + 红色边框+错误提示
- 单位切换：输入框右侧单位下拉（支持一键切换国际/英制）
- 参数联动：修改参数后自动刷新预览（防抖300ms）

## 七、优先级实施计划

### Phase 1（优先级最高）
1. 统一配色+字体系统（CSS变量）
2. 顶部标题栏重构
3. 左侧导航栏重设计（加激活态）
4. 首页改版（项目卡片化）

### Phase 2
5. 仿真模块标签页化
6. 材料/边界条件面板重设计
7. 右侧参数面板规范化

### Phase 3
8. 后处理全屏+图表联动
9. 笔记模块重构
10. 动效和过渡优化
