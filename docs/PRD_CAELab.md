## 1. 产品定位与愿景

### 1.1 产品定位升级

| 维度 | 旧版定位 | 新版定位 |
|------|---------|---------|
| **产品名称** | 云CAE / 独立CAE仿真平台 | **CAELab** - 科研与工程创作一体化工作台 |
| **核心定位** | 独立CAE仿真工具 | **创作驱动**的科研与工程一体化平台 |
| **差异化** | 轻量化云端仿真 | 仿真即创作，创作即记录，记录即知识 |

### 1.2 产品愿景

> **"让CAE仿真成为科研创作的自然延伸，而非孤立的分析工具"**

- 学生、科研人员、工程师在CAELab中不仅能做仿真，还能**记录思路、整理笔记、编写代码、沉淀知识**
- 仿真结果与笔记、代码、3D模型形成**有机整体**，而非孤立的数据文件
- 打破工具割裂：不再需要在仿真软件、笔记软件、代码编辑器之间反复切换

### 1.3 目标用户

| 用户群体 | 核心诉求 | CAELab如何满足 |
|---------|---------|----------------|
| 高校理工科学生 | 课程学习、毕业设计、竞赛 | 笔记+仿真+代码，一站式完成研究过程 |
| 科研人员 | 论文写作、实验记录、数据整理 | 仿真结果内嵌笔记，LaTeX公式支持 |
| 工程设计师 | 快速仿真、快速迭代、快速汇报 | 仿真+可视化+报告，效率提升 |
| 中小企业工程师 | 按需仿真、成本可控 | 云端弹性计算，按需付费 |

## 3. 笔记创作模块需求（新增）

### 3.1 模块定位

笔记模块是CAELab的**创作中枢**，区别于传统笔记本：
- 不仅记录文字，还能嵌入**活的数据**（3D模型、仿真结果、代码片段）
- 嵌入的内容**双向关联**，点击即可跳转到对应工具

### 3.2 功能列表

#### 3.2.1 基础编辑能力

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 富文本编辑 | 支持标题、列表、引用、表格等标准格式 | P0 |
| 手写输入 | 支持Apple Pencil、触控笔手写，笔迹识别与导出 | P0 |
| LaTeX公式 | 完整LaTeX支持，行内公式+独立公式块 | P0 |
| Markdown兼容 | 支持Markdown语法，与LaTeX混排 | P1 |
| 双向链接 | 笔记之间相互引用，形成知识网络 | P1 |

#### 3.2.2 内嵌能力

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 内嵌3D模型 | 在笔记中嵌入交互式3D视图，点击跳转到建模模块 | P0 |
| 内嵌代码块 | 支持Python/C++/MATLAB等语言语法高亮，可执行 | P0 |
| 内嵌仿真结果 | 粘贴仿真云图、曲线图，点击展开详情 | P0 |
| 内嵌计算器 | 快捷插入计算结果，实时更新 | P1 |
| 内嵌图表 | 从代码/仿真结果生成图表嵌入笔记 | P1 |

#### 3.2.3 知识管理

| 功能 | 描述 | 优先级 |
|------|------|--------|
| 笔记本分类 | 文件夹/标签体系管理笔记 | P0 |
| 全文搜索 | 搜索笔记标题、内容、嵌入的代码 | P0 |
| 版本历史 | 笔记编辑记录，支持回滚 | P1 |
| 模板市场 | 预设课程笔记模板、实验报告模板、仿真报告模板 | P2 |

### 3.3 手写功能详细需求

- **输入设备支持**：Apple Pencil、Samsung S Pen、Surface Pen、触控板手写
- **笔迹处理**：
  - 压感支持（笔触粗细、透明度）
  - 橡皮擦（精确擦除/划线擦除）
  - 套索选择与移动
- **手写识别**：
  - 手写转LaTeX公式
  - 手写转文字（可选功能）
- **导出选项**：手写笔记导出为PNG/SVG/PDF

### 3.4 LaTeX支持详细需求

- **编辑模式**：
  - 行内公式：`$E=mc^2$`
  - 独立公式块：`$$\int_0^1 f(x)dx$$`
  - 公式块编号与引用
- **实时预览**：边输入边渲染，支持复杂公式（矩阵、分段函数、积分等）
- **公式库**：常用公式模板快捷插入
- **交叉引用**：公式编号与论文引用格式兼容

## 4. 多工具联动需求（新增）

### 4.1 联动架构

多工具联动是CAELab的核心竞争力，打破传统工具割裂：

```
        ┌───────────────┐
        │   📝 笔记     │◄──────────────────────────┐
        │  (创作中枢)   │                           │
        └───────┬───────┘                           │
                │                                   │
    ┌───────────┼───────────┐                       │
    ▼           ▼           ▼                       │
┌─────────┐ ┌─────────┐ ┌─────────┐                │
│ 🎨 建模 │ │ 💻 代码 │ │ 🔬 仿真 │                │
└────┬────┘ └────┬────┘ └────┬────┘                │
     │           │           │                      │
     └───────────┴───────────┘                      │
                │                                   │
                └──────────────────────────────────┘
                       双向跳转与数据互通
```

### 4.2 联动场景

#### 场景1：笔记 ↔ 建模

| 操作方向 | 描述 |
|---------|------|
| 笔记 → 建模 | 在笔记中点击3D模型缩略图，跳转到建模模块并打开对应模型 |
| 建模 → 笔记 | 建模完成后，一键将模型快照插入当前笔记 |
| 快捷备注 | 建模时添加文字标注，自动同步到关联笔记 |

#### 场景2：笔记 ↔ 代码

| 操作方向 | 描述 |
|---------|------|
| 笔记 → 代码 | 点击代码块，展开完整代码编辑器，修改后同步回笔记 |
| 代码 → 笔记 | 代码运行结果（图表、数据）一键插入当前笔记 |
| 注释同步 | 代码中的注释可选择性同步为笔记内容 |

#### 场景3：笔记 ↔ 仿真

| 操作方向 | 描述 |
|---------|------|
| 笔记 → 仿真 | 点击仿真结果卡片，跳转到仿真模块查看完整分析结果 |
| 仿真 → 笔记 | 仿真完成后，可选择将关键云图、曲线图、关键数据一键插入笔记 |
| 报告生成 | 笔记中的仿真结果自动汇总，生成标准仿真报告 |

#### 场景4：建模 ↔ 代码 ↔ 仿真

| 操作方向 | 描述 |
|---------|------|
| 建模 → 代码 | 几何参数自动生成到代码模板（如参数化建模脚本） |
| 代码 → 建模 | 代码计算的几何参数更新到建模模块 |
| 代码 → 仿真 | 代码输出的物理参数自动配置到仿真边界条件 |
| 仿真 → 代码 | 仿真结果数据可被代码模块读取用于后处理 |

### 4.3 联动技术要求

| 要求 | 描述 |
|------|------|
| 实时同步 | 联动数据变更后，相关模块实时感知并更新 |
| 上下文保持 | 跳转时保持当前工作上下文（如选中的几何、当前行等） |
| 回退机制 | 跳转后返回时，恢复跳转前的编辑状态 |
| 冲突处理 | 多处编辑同一数据时，提示冲突并允许用户选择 |

## 6. 用户流程

### 6.1 主流程（升级版）

```
┌──────────────────────────────────────────────────────────────────┐
│                         CAELab 主流程                            │
├──────────────────────────────────────────────────────────────────┤
│                                                                  │

│                    │                                              │

│        ▼           ▼           ▼                                 │
│   📝笔记优先    🎨建模优先    🔬仿真优先                         │
│   (研究记录)    (几何创建)    (直接分析)                         │
│        │           │           │                                 │
│        └───────────┼───────────┘                                 │
│                    ▼                                              │
│            💻代码编写/调试                                         │
│                    │                                              │
│                    ▼                                              │
│            🔬仿真计算与分析                                        │
│                    │                                              │
│                    ▼                                              │
│            📊结果可视化+报告                                       │
│                    │                                              │
│                    ▼                                              │
│            📝整理笔记与知识沉淀                                    │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### 6.2 详细流程

#### 6.2.1 笔记创作流程

1. **新建笔记**
   - 进入笔记本列表
   - 新建或选择笔记本
   - 选择笔记模板（空白/课程笔记/实验报告/仿真报告）

2. **编辑内容**
   - 使用富文本编辑器
   - 手写输入（平板端）
   - 插入LaTeX公式

3. **内嵌对象**
   - 插入3D模型：选择已有模型或新建
   - 插入代码：选择语言，编写代码，可执行
   - 插入仿真结果：从已完成项目中选择

4. **保存与分享**
   - 保存笔记
   - 导出PDF/Markdown
   - 分享给协作者

#### 6.2.2 多工具联动流程

1. **从笔记跳转**
   - 点击内嵌对象（模型/代码/结果）
   - 系统打开对应模块，加载对应内容
   - 编辑完成后，点击"返回笔记"恢复上下文

2. **插入到笔记**
   - 在建模/代码/仿真模块中
   - 点击"插入到笔记"按钮
   - 选择目标笔记和插入位置

3. **双向同步**
   - 修改模型/代码/结果后
   - 笔记中的嵌入对象自动更新
   - 笔记中的说明文字保持手动编辑

### 6.3 仿真分析流程（保留）

1. **项目创建**
   - 点击"新建项目"
   - 输入项目名称、选择仿真类型
   - 进入项目工作空间

2. **几何处理**
   - 选择：新建几何 / 导入外部文件 / 从笔记导入
   - 进行几何清理和修复
   - 确认几何模型

3. **网格划分**
   - 设置网格基本参数
   - 执行自动划分
   - 检查网格质量
   - 如有必要进行局部调整
   - 确认网格

4. **物理场设置**
   - 选择仿真类型（结构/流体/热/耦合）
   - 添加材料属性
   - 设置边界条件
   - 设置荷载条件
   - 确认所有设置

5. **求解计算**
   - 设置求解控制参数
   - 预估计算资源和费用
   - 确认提交
   - 支付（扣除相应点数）
   - 等待计算完成（可实时查看进度）

6. **结果后处理**
   - 自动加载结果
   - 选择感兴趣的结果变量
   - 生成云图/矢量图
   - 提取关键数据曲线
   - 生成动画
   - 导出数据或报告
   - **一键插入笔记**（新增）

## 9. 非功能性需求

### 9.1 性能需求

| 指标 | 要求 |
|------|------|
| 网格划分 | 支持最大1000万单元 |
| 结果可视化 | 渲染流畅（<=100万单元实时交互） |
| 网页加载 | < 3秒 |
| 计算排队 | 不超过30分钟（非峰值） |
| 笔记同步 | 编辑后<1秒同步到云端 |
| 工具跳转 | 跳转响应<500ms |

### 9.2 安全需求

- 用户数据加密存储
- 项目数据隔离
- 支付信息通过第三方处理
- 定期数据备份

### 9.3 兼容性

- **桌面端**：macOS、Windows、Linux（优先）
- **平板端**：iPad（优先）、Android平板（Phase 2）
- **移动端**：iOS/Android手机（Phase 3）
- 支持主流浏览器（Chrome、Firefox、Edge、Safari）
