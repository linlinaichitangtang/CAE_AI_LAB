# 独立CAE项目需求文档

## 1. 产品定位与核心需求

### 1.1 产品定位
**独立CAE（计算机辅助工程）仿真平台** - 面向工程师、科研人员和学生的轻量化、云端化CAE仿真分析工具，提供结构力学、流体力学、热传导等多物理场仿真能力，降低CAE使用门槛，提升工程设计效率。

### 1.2 核心需求
1. **云端化仿真**：用户无需安装大型本地软件，通过浏览器即可完成完整仿真流程
2. **轻量化体验**：简化传统CAE软件复杂操作流程，提供智能化建模和网格划分
3. **按需付费**：按计算资源使用量计费，降低个人和中小企业使用成本
4. **协同分享**：支持仿真项目多人协作和结果分享，便于团队协作
5. **跨设备访问**：支持PC、平板、手机多端访问，随时随地查看仿真结果

### 1.3 目标用户
- 机械/土木/航空航天等领域的工程师
- 高校和科研机构的科研人员
- 相关专业的学生
- 中小企业CAE分析需求团队

## 3. 用户流程

### 3.1 主流程
```
用户登录 → 创建新项目 → 导入/创建几何模型 → 网格划分 → 设置物理场 → 提交计算 → 查看结果 → 导出报告/数据
```

### 3.2 详细流程

#### 3.2.1 首次用户流程
1. 用户注册账号
2. 邮箱/手机验证
3. 完善个人信息
4. 选择套餐/充值
5. 查看新手引导
6. 创建第一个项目

#### 3.2.2 仿真分析流程
1. **项目创建**
   - 点击"新建项目"
   - 输入项目名称、选择仿真类型
   - 进入项目工作空间

2. **几何处理**
   - 选择：新建几何 / 导入外部文件
   - 进行几何清理和修复
   - 确认几何模型

3. **网格划分**
   - 设置网格基本参数
   - 执行自动划分
   - 检查网格质量
   - 如有必要进行局部调整
   - 确认网格

4. **物理场设置**
   - 选择仿真类型（结构/流体/热/耦合）
   - 添加材料属性
   - 设置边界条件
   - 设置荷载条件
   - 确认所有设置

5. **求解计算**
   - 设置求解控制参数
   - 预估计算资源和费用
   - 确认提交
   - 支付（扣除相应点数）
   - 等待计算完成（可实时查看进度）

6. **结果后处理**
   - 自动加载结果
   - 选择感兴趣的结果变量
   - 生成云图/矢量图
   - 提取关键数据曲线
   - 生成动画
   - 导出数据或报告

### 3.3 协作流程
1. 项目创建者点击"分享"
2. 输入协作者邮箱/用户名
3. 设置权限（查看/编辑）
4. 协作者收到邀请通知
5. 协作者接受邀请后可进入项目协作
6. 多人可同时查看项目，依次编辑

## 5. 非功能性需求

### 5.1 性能需求
- 网格划分支持最大1000万单元
- 结果可视化渲染流畅（<=100万单元实时交互）
- 网页加载时间 < 3秒
- 计算任务排队时间不超过30分钟（非峰值）

### 5.2 安全需求
- 用户数据加密存储
- 项目数据隔离，不同用户不可互相访问
- 支付信息不存储，通过第三方支付平台处理
- 定期数据备份

### 5.3 兼容性
- 支持主流浏览器（Chrome、Firefox、Edge、Safari）
- 响应式设计，支持不同分辨率屏幕
