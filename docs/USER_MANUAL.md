## 1. 简介

### 1.1 CAELab 是什么

CAELab 是一款**科研与工程创作一体化工作台**，旨在打破传统工具之间的割裂。它将 CAE 仿真分析、3D 几何建模、富文本笔记创作和代码编辑整合在同一个桌面应用中，实现"仿真即创作、创作即记录、记录即知识"的工作流程。

### 1.2 核心功能概览

| 功能模块 | 说明 |
|---------|------|
| **仿真分析** | 支持结构分析、模态分析、屈曲分析、热传导分析、瞬态动力学、CFD 流场分析、接触分析、拓扑优化等多种分析类型 |
| **3D 建模** | 基本几何体创建、CSG 布尔运算、STEP/IGES 文件导入、场景树管理 |
| **笔记创作** | TipTap 富文本编辑、手写输入、LaTeX 公式、嵌入仿真结果和 3D 模型 |
| **代码编辑** | Monaco Editor 代码编辑器，支持 Python 脚本编辑、语法高亮、脚本自动化 |

### 1.3 系统要求

| 项目 | 最低要求 |
|------|---------|
| 操作系统 | macOS 10.15+ / Windows 10+ / Linux (x64) |
| 内存 | 4 GB RAM（推荐 8 GB） |
| 磁盘空间 | 500 MB（不含求解器） |
| CAE 求解器 | CalculiX（可选，用于真实求解） |

> **说明**：CAELab 可在没有 CalculiX 的情况下运行，内置演示求解器可完成基本仿真。安装 CalculiX 后可获得更高精度的求解结果。

### 1.4 安装与启动

1. 从官网下载对应平台的安装包：
   - **macOS**：`.dmg` 格式，拖拽到 Applications 文件夹
   - **Windows**：`.exe` 格式（NSIS 安装器），按提示安装
   - **Linux**：`.deb` 或 `.AppImage` 格式
2. 双击启动 CAELab
3. 首次启动将显示 4 步新手引导，引导您了解核心功能
4. 引导完成后进入首页，即可开始使用

## 2. 快速入门（30分钟完成第一个仿真）

本节通过一个悬臂梁静力分析案例，带您快速了解 CAELab 的完整仿真流程。

### 2.1 创建项目

1. 在首页点击 **"新建项目"** 按钮
2. 输入项目名称（如"悬臂梁分析"）和描述
3. 点击确认，项目创建成功
4. 在首页项目卡片中点击刚创建的项目，进入项目工作区

### 2.2 生成网格

1. 在左侧导航栏点击 **仿真分析** 图标，进入仿真视图
2. 在右侧面板的 **网格** 选项卡中：
   - **选择维度**：选择 `3D`（三维分析）
   - **设置几何参数**：
     - 长度 (X)：`1.0` m
     - 宽度 (Y)：`0.1` m
     - 高度 (Z)：`0.1` m
   - **设置网格密度**：
     - X 方向分割数：`20`
     - Y 方向分割数：`2`
     - Z 方向分割数：`2`
   - **选择单元类型**：`C3D8`（六面体 8 节点单元）
3. 点击 **"生成网格"** 按钮
4. 3D 视图中将显示生成的网格模型

### 2.3 定义材料

1. 切换到右侧面板的 **材料** 选项卡
2. **从材料库选择**：
   - 点击"材料库"按钮，浏览 50+ 预置材料
   - 搜索或筛选找到"钢 (Steel)"
   - 点击选择，材料参数自动填入
3. **或手动输入参数**：
   - 材料名称：`钢`
   - 弹性模量 E：`200000` MPa
   - 泊松比：`0.3`
   - 密度：`7.85e-9` ton/mm^3

### 2.4 设置边界条件

1. 切换到右侧面板的 **边界条件** 选项卡
2. **添加固定约束**：
   - 约束类型选择 `Fixed`（全固定）
   - 选择约束面：`X 最小面`（左端面）
   - 点击"添加约束"
3. **添加载荷**：
   - 载荷类型选择 `集中力`
   - 施加位置：`X 最大面顶部`
   - 载荷大小：`1000` N
   - 载荷方向：`Y` 方向
   - 点击"添加载荷"

### 2.5 运行求解

1. 在顶部工具栏的 **分析类型** 下拉框中选择 `结构分析`
2. 点击 **"运行求解器"** 按钮
3. 观察求解进度：
   - 进度条显示实时求解进度百分比
   - 显示预计剩余时间 (ETA)
   - 可点击 **"取消"** 按钮中止求解
   - 点击 **"计算日志"** 查看详细求解输出
4. 求解完成后，进度条变为绿色并显示总耗时

### 2.6 查看结果

1. 求解完成后，3D 视图自动显示结果云图
2. **切换结果显示**：
   - 位移云图：显示节点位移幅值
   - 应力云图：显示应力分量 (Sxx, Syy, Szz, Sxy, Syz, Sxz)
   - Von Mises 云图：显示等效应力
3. **变形显示**：调节变形放大比例，查看结构变形趋势
4. **剖切面工具**：
   - 勾选"剖切面"开关
   - 选择剖切方向 (XY/XZ/YZ)
   - 拖动滑块调整剖切位置，查看内部截面应力分布
5. **变形动画**（瞬态分析时可用）：
   - 点击播放按钮回放变形过程
   - 调整播放速度 (0.25x / 0.5x / 1x / 2x)

### 2.7 导出报告

1. 点击顶部工具栏的 **"生成报告"** 按钮
2. 选择报告格式：
   - **HTML 报告**：在浏览器中打开，包含完整的模型参数和结果数据
   - **PDF 报告**：生成 PDF 文件，适合打印和分享（基于 jsPDF）

## 3. 建模功能

### 3.1 基本几何体

CAELab 支持创建以下基本几何体：

| 几何体 | 说明 |
|--------|------|
| 长方体 (Box) | 通过长、宽、高定义 |
| 圆柱体 (Cylinder) | 通过半径和高度定义 |
| 球体 (Sphere) | 通过半径定义 |
| 圆锥体 (Cone) | 通过底部半径和高度定义 |

**操作步骤**：
1. 进入 **3D 建模** 视图
2. 点击顶部 **"添加几何体"** 按钮
3. 选择几何体类型
4. 设置几何参数
5. 几何体将出现在 3D 视图和左侧场景树中

### 3.2 CSG 布尔运算

CSG（Constructive Solid Geometry）布尔运算用于组合基本几何体创建复杂形状：

| 运算类型 | 说明 |
|---------|------|
| 并集 (Union) | 合并两个几何体 |
| 差集 (Subtract) | 从一个几何体中减去另一个 |
| 交集 (Intersect) | 保留两个几何体的重叠部分 |

**操作步骤**：
1. 在场景树中选择两个几何体
2. 右键点击，选择布尔运算类型
3. 运算结果将替换原始几何体

### 3.3 STEP/IGES 文件导入

CAELab 支持导入外部 CAD 模型文件。

**支持的格式**：
- STEP (.step, .stp) -- 推荐格式
- IGES (.iges, .igs)

**前置要求**（需安装以下工具之一）：
- **GMSH**（推荐）：[https://gmsh.info](https://gmsh.info)
- **FreeCAD**：[https://freecad.org](https://freecad.org)

**导入步骤**：
1. 进入 **3D 建模** 视图
2. 点击顶部 **"导入"** 按钮
3. 选择 STEP/IGES 文件
4. 等待转换完成（系统自动调用 GMSH 或 FreeCAD）
5. 导入的网格将自动加载到建模视图和仿真模块

### 3.4 几何体管理

左侧场景树提供完整的几何体管理功能：

- **选择**：单击场景树中的几何体名称选中，3D 视图中高亮显示
- **显示/隐藏**：点击眼睛图标切换几何体可见性
- **删除**：选中后按 Delete 键或右键删除
- **属性编辑**：选中后在右侧面板中修改几何参数

## 4. 仿真分析详解

### 4.1 结构分析（静力）

线性静力分析用于计算结构在静态载荷下的位移、应力和应变。

**适用场景**：梁的弯曲、板的受力、支架强度校核等。

**操作步骤**：
1. 生成网格并定义材料
2. 设置边界条件（固定约束 + 载荷）
3. 分析类型选择 `结构分析`
4. 运行求解
5. 查看位移、应力、Von Mises 等结果

### 4.2 模态分析

模态分析用于计算结构的固有频率和振型。

**适用场景**：振动特性评估、共振风险分析。

**操作步骤**：
1. 生成网格并定义材料（需包含密度）
2. 设置固定约束（无需载荷）
3. 分析类型选择 `模态分析`
4. 设置求解模态阶数（默认 6 阶）
5. 运行求解
6. 查看各阶固有频率和对应振型动画

### 4.3 屈曲分析

屈曲分析用于计算结构的临界屈曲载荷。

**适用场景**：细长压杆、薄壁结构的稳定性评估。

**操作步骤**：
1. 完成静力分析的前处理设置
2. 分析类型选择 `屈曲分析`
3. 运行求解
4. 查看屈曲因子和屈曲模态

### 4.4 热传导分析

热传导分析用于计算温度场分布和热流。

**适用场景**：散热设计、热应力分析。

**操作步骤**：
1. 生成网格并定义材料（需包含热导率）
2. 设置温度边界条件或热流载荷
3. 分析类型选择 `热传导分析`
4. 运行求解
5. 查看温度分布云图和热流矢量

### 4.5 瞬态动力学

瞬态动力学分析用于计算结构在随时间变化载荷下的动态响应。

**适用场景**：冲击载荷、振动响应、爆炸分析。

**操作步骤**：
1. 生成网格并定义材料（需包含密度）
2. 设置边界条件和时变载荷
3. 分析类型选择 `瞬态动力学`
4. 设置时间步长和总时间
5. 运行求解
6. 使用变形动画回放各时间步结果

### 4.6 CFD 流场分析

CFD 流场分析用于计算流体流动特性。

**适用场景**：管道流动、外流场分析、通风设计。

**操作步骤**：
1. 生成流体域网格
2. 定义流体材料属性
3. 设置进出口边界条件
4. 分析类型选择 `CFD 流场分析`
5. 运行求解
6. 查看速度场、压力场云图和流线

### 4.7 接触分析

接触分析用于模拟多个部件之间的接触行为。

**支持的接触类型**：

| 类型 | 说明 |
|------|------|
| 面对面 (S2S) | Surface-to-Surface 接触 |
| 点对面 (N2S) | Node-to-Surface 接触 |
| 绑定 (Tie) | 将两个面绑定在一起 |
| 螺栓预紧力 | 模拟螺栓连接 |

**操作步骤**：
1. 创建包含多个部件的模型
2. 在仿真视图中切换到 **接触** 选项卡
3. 定义接触对（主面 + 从面）
4. 设置接触参数（摩擦系数、接触刚度等）
5. 运行求解
6. 查看接触压力和接触状态

### 4.8 拓扑优化

拓扑优化基于 SIMP 方法，在给定设计空间内寻找最优材料分布。

**操作步骤**：
1. 进入 **拓扑优化** 视图
2. 定义设计空间（保留区域和优化区域）
3. 设置优化参数：
   - 体积约束（目标保留体积比例）
   - 位移约束
   - 惩罚因子
4. 运行优化迭代
5. 查看密度分布结果和优化后的结构形状

## 5. 网格技术

### 5.1 结构化网格生成

CAELab 支持矩形/长方体区域的结构化网格生成。

**操作步骤**：
1. 在仿真视图的网格选项卡中设置参数
2. 选择维度（2D 或 3D）
3. 输入几何尺寸
4. 设置各方向的分割数
5. 选择单元类型
6. 点击"生成网格"

### 5.2 局部网格加密

在应力集中区域（如孔洞、缺口、载荷施加点）使用局部加密功能，提高关键区域的计算精度。

**操作步骤**：
1. 生成基础网格后，点击 **"局部加密"** 按钮
2. 选择加密区域类型：`边` / `面` / `体`
3. 指定加密方向和范围
4. 设置加密比例（1.5x ~ 4.0x）
5. 系统自动生成渐变过渡的加密网格

### 5.3 网格质量检查

生成网格后，使用网格质量检查功能评估网格质量，确保计算精度。

**质量指标**：

| 指标 | 含义 | 建议值 |
|------|------|--------|
| 长宽比 (Aspect Ratio) | 单元最长边与最短边之比 | < 5 |
| 雅可比比 (Jacobian) | 单元映射质量度量 | > 0.5 |
| 偏斜度 (Skewness) | 单元偏离理想形状的程度 | < 0.5 |
| 翘曲角 (Warp Angle) | 四边形的翘曲程度 | < 15 度 |

**操作步骤**：
1. 生成网格后，点击 **"网格质量检查"** 按钮
2. 查看总体评分（平均质量百分比）
3. 查看质量分布柱状图（优秀/良好/一般/较差）
4. 根据检查结果调整网格参数后重新生成

## 6. 后处理

### 6.1 云图显示

求解完成后，CAELab 提供多种结果云图：

| 结果类型 | 说明 |
|---------|------|
| 位移 (Displacement) | 节点位移幅值 |
| 应力分量 (Stress) | Sxx, Syy, Szz, Sxy, Syz, Sxz |
| Von Mises 等效应力 | 综合应力评估指标 |
| 温度场 (Thermal) | 温度分布（热传导分析） |

**操作**：在结果面板的下拉菜单中选择要显示的结果类型。

### 6.2 色标与颜色映射

- 云图右侧显示色标条，标注最小值和最大值
- 支持多种颜色映射方案（彩虹色、热力图、灰度等）
- 可自定义色标范围

### 6.3 剖切面工具

剖切面工具用于查看模型内部截面的结果分布。

**操作步骤**：
1. 在 3D 视图右上角勾选 **"剖切面"** 开关
2. 选择剖切方向：`XY` / `XZ` / `YZ`
3. 拖动滑块调整剖切位置
4. 截面处将显示内部应力/位移分布

### 6.4 变形动画

对于瞬态分析和模态分析，可播放多时间步/多阶模态的动画。

**操作步骤**：
1. 在 3D 视图底部找到动画控制栏
2. 点击 **播放/暂停** 按钮
3. 拖动进度条跳转到特定帧
4. 选择播放速度：`0.25x` / `0.5x` / `1x` / `2x`
5. 帧信息显示当前帧号和对应时间

### 6.5 数据导出

- **导出图片**：将当前 3D 视图截图保存为图片文件
- **导出 HTML 报告**：包含完整模型参数和结果数据
- **导出 PDF 报告**：适合打印和正式提交

## 7. 笔记系统

### 7.1 富文本编辑

CAELab 使用 TipTap 编辑器提供强大的富文本编辑功能：

- 文本格式：加粗、斜体、下划线、删除线
- 标题层级：H1 ~ H3
- 列表：有序列表、无序列表
- 引用块、代码块
- 表格插入和编辑

### 7.2 手写输入

- 支持手写板/触控笔输入
- 手写内容可转换为文本
- 手写画布可嵌入笔记中

### 7.3 LaTeX 公式

CAELab 使用 KaTeX 渲染引擎支持 LaTeX 数学公式：

- 行内公式：使用 `$...$` 语法
- 独立公式块：使用 `$$...$$` 语法
- 提供公式编辑器辅助输入

**示例**：

```
行内公式：$E = mc^2$

独立公式：
$$\sigma = \frac{F}{A}$$
```

### 7.4 嵌入仿真结果

在笔记中嵌入仿真结果，实现分析与记录的无缝联动：

1. 在仿真视图中完成求解后，点击 **"嵌入到笔记"** 按钮
2. 选择要嵌入的内容（云图截图、结果数据表格等）
3. 嵌入的内容将显示在笔记中，并保持与仿真模块的链接
4. 点击嵌入卡片可跳转回仿真视图

### 7.5 嵌入 3D 模型

在笔记中嵌入 3D 几何模型：

1. 在建模视图中点击 **"嵌入到笔记"** 按钮
2. 选择要嵌入的几何体
3. 嵌入的 3D 模型可在笔记中交互式旋转和缩放

## 8. 代码编辑器

### 8.1 Python 脚本编辑

CAELab 内置基于 Monaco Editor 的代码编辑器：

- 支持 Python 脚本编辑和运行
- 文件资源管理器管理脚本文件

## 11. 常见问题 (FAQ)

### CalculiX 未安装怎么办？

CAELab 可以在没有 CalculiX 的情况下运行。内置演示求解器可完成基本仿真分析。如需使用 CalculiX 获得更高精度：

1. 访问 [CalculiX 官网](http://www.calculix.de) 下载安装
2. 确保 `ccx` 命令在系统 PATH 中可用
3. 在终端中运行 `ccx` 验证安装成功

### STEP 导入失败怎么办？

STEP 文件导入需要 GMSH 或 FreeCAD：

1. 确认已安装 GMSH 或 FreeCAD
2. 确认 GMSH/FreeCAD 可执行文件在系统 PATH 中
3. 检查 STEP 文件是否损坏（尝试在其他 CAD 软件中打开）
4. 如果模型过于复杂，尝试简化后再导入

### 求解不收敛怎么办？

求解不收敛的常见原因和解决方法：

| 原因 | 解决方法 |
|------|---------|
| 网格质量差 | 运行网格质量检查，加密关键区域 |
| 载荷过大 | 减小载荷步长，使用增量加载 |
| 材料参数不合理 | 检查材料参数的单位和数量级 |
| 边界条件不足 | 确保模型没有刚体位移（缺少约束） |
| 接触设置不当 | 调整接触刚度和穿透容差 |

### 内存不足怎么办？

- 减少网格密度（降低分割数）
- 使用局部加密代替全局加密
- 关闭其他占用内存的应用程序
- 增加系统虚拟内存/交换空间

## 附录 C：理论公式参考

### C.1 梁弯曲理论

**悬臂梁 - 端点集中载荷**：

$$\delta = \frac{PL^3}{3EI}$$

$$\sigma_{max} = \frac{Mc}{I} = \frac{FL(h/2)}{I}$$

**悬臂梁 - 均布载荷**：

$$\delta = \frac{qL^4}{8EI}$$

$$\sigma_{max} = \frac{qL^2 c}{2I}$$

**简支梁 - 中心集中载荷**：

$$\delta = \frac{FL^3}{48EI}$$

$$\sigma_{max} = \frac{FLc}{4I}$$

其中：
- $P$ = 集中力 (N)
- $q$ = 均布载荷线密度 (N/m)
- $L$ = 梁长度 (m)
- $E$ = 弹性模量 (Pa)
- $I$ = 截面惯性矩 (m^4)，矩形截面 $I = bh^3/12$
- $c$ = 截面中性轴到最外缘距离 (m)，$c = h/2$
- $b$ = 截面宽度 (m)
- $h$ = 截面高度 (m)

### C.2 Von Mises 等效应力

$$\sigma_{VM} = \sqrt{\frac{1}{2}\left[(\sigma_1 - \sigma_2)^2 + (\sigma_2 - \sigma_3)^2 + (\sigma_3 - \sigma_1)^2\right]}$$

### C.3 固有频率

单自由度系统固有频率：

$$f = \frac{1}{2\pi}\sqrt{\frac{k}{m}}$$

悬臂梁第一阶固有频率：

$$f_1 = \frac{1.875^2}{2\pi L^2}\sqrt{\frac{EI}{\rho A}}$$

其中 $\rho$ 为材料密度，$A$ 为截面积。
