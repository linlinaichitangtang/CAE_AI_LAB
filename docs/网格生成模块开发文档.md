# 结构化网格划分模块 - 开发文档

## 任务概述

实现二维/三维结构化网格生成模块，为CAE分析提供高质量的网格数据。

## 功能特性

### 1. 网格生成功能

| 功能 | 描述 | 元素类型 |
|------|------|----------|
| 2D矩形网格 | 生成二维矩形区域的结构化网格 | S4, S8, S3, S6 |
| 3D长方体网格 | 生成三维长方体区域的结构化网格 | C3D8, C3D20, C3D27, C3D4, C3D10, C3D6, C3D15 |

### 2. 数据结构

#### StructuredMesh
```rust
pub struct StructuredMesh {
    pub dimensions: (usize, usize, usize),  // (nx, ny, nz)
    pub size: (f64, f64, f64),                 // 物理尺寸
    pub nodes: Vec<Vec<f64>>,                  // 节点坐标 [node_id][x, y, z]
    pub elements: Vec<Vec<usize>>,             // 单元连接 [elem_id][node1, node2, ...]
    pub element_type: String,                  // 元素类型标识
    pub num_nodes: usize,                      // 节点总数
    pub num_elements: usize,                  // 单元总数
}
```

#### GridConfig
```rust
pub struct GridConfig {
    pub nx: usize,              // X方向单元数
    pub ny: usize,              // Y方向单元数
    pub nz: usize,              // Z方向单元数 (1=2D)
    pub size_x: f64,            // X方向物理尺寸
    pub size_y: f64,            // Y方向物理尺寸
    pub size_z: f64,            // Z方向物理尺寸 (0=2D)
    pub element_type: MeshElementType,  // 元素类型
}
```

#### MeshElementType 枚举
```rust
// 2D元素
Quad4,   // 4节点四边形 (S4)
Quad8,   // 8节点四边形 (S8)
Tri3,    // 3节点三角形 (S3)
Tri6,    // 6节点三角形 (S6)

// 3D元素
Hex8,    // 8节点六面体 (C3D8)
Hex20,   // 20节点六面体 (C3D20)
Hex27,   // 27节点六面体 (C3D27)
Tet4,    // 4节点四面体 (C3D4)
Tet10,   // 10节点四面体 (C3D10)
Wedge6,  // 6节点楔形体 (C3D6)
Wedge15, // 15节点楔形体 (C3D15)
```

### 3. API接口

#### Rust API (commands/solver.rs)

```rust
// 通过 CalculiXSolver 提供以下方法:

/// 生成2D矩形网格
pub fn generate_2d_mesh(
    nx: usize, ny: usize,
    size_x: f64, size_y: f64
) -> Result<StructuredMesh, MeshError>

/// 生成3D长方体网格
pub fn generate_3d_mesh(
    nx: usize, ny: usize, nz: usize,
    size_x: f64, size_y: f64, size_z: f64
) -> Result<StructuredMesh, MeshError>

/// 生成自定义配置的结构化网格
pub fn generate_structured_mesh(
    config: GridConfig
) -> Result<StructuredMesh, MeshError>
```

#### Tauri Commands (commands/cae_api.rs)

| 命令 | 参数 | 返回值 | 说明 |
|------|------|--------|------|
| `generate_2d_mesh` | nx, ny, size_x, size_y | StructuredMesh | 生成2D网格 |
| `generate_3d_mesh` | nx, ny, nz, size_x, size_y, size_z | StructuredMesh | 生成3D网格 |
| `generate_structured_mesh` | nx, ny, nz, size_x, size_y, size_z, element_type | StructuredMesh | 自定义网格 |
| `export_mesh_to_inp` | mesh, material_name, output_path | String | 导出为INP文件 |

### 5. 网格生成算法

#### 2D矩形网格
```
节点编号 (3x2 网格示例):
  6 --- 7 --- 8
  |  3  |  4  |
  3 --- 4 --- 5
  |  1  |  2  |
  0 --- 1 --- 2

节点坐标: (i*dx, j*dy, 0)
单元节点: [n0, n1, n2, n3]
```

#### 3D长方体网格
```
单元节点 (Hex8):
      3 -------- 7
     /|         /|
    / |        / |
   1 --------5   |
   |  2-------|-6
   | /        | /
   |/         |/
   0 --------4

节点编号使用offset排序 (k*nx*ny + j*nx + i)
保证更好的缓存性能
```

## 使用示例

### 前端调用示例

```typescript
// 生成2D网格
const mesh2d = await invoke('generate_2d_mesh', {
    nx: 10,
    ny: 5,
    size_x: 100.0,
    size_y: 50.0
});

// 生成3D网格
const mesh3d = await invoke('generate_3d_mesh', {
    nx: 10,
    ny: 10,
    nz: 5,
    size_x: 100.0,
    size_y: 100.0,
    size_z: 50.0
});

// 自定义元素类型
const customMesh = await invoke('generate_structured_mesh', {
    nx: 10,
    ny: 10,
    nz: 5,
    size_x: 100.0,
    size_y: 100.0,
    size_z: 50.0,
    element_type: "C3D20"
});

// 导出到INP文件
const outputPath = await invoke('export_mesh_to_inp', {
    mesh: mesh3d,
    material_name: "Steel",
    output_path: "/path/to/model.inp"
});
```

### 后端使用示例

```rust
use crate::commands::solver::{CalculiXSolver, GridConfig, MeshElementType};

let config = GridConfig::new_2d(10, 5, 100.0, 50.0);
let mesh = CalculiXSolver::generate_2d_mesh(10, 5, 100.0, 50.0)?;

// 转换为 INP 生成器格式
let (nodes, elements) = mesh_to_nodes_elements(&mesh);
```
