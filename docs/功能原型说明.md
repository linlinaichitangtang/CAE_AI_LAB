## 交互流程原型

1. **用户登录** → **项目列表** → **新建项目向导** → **仿真工作区**
2. 每一步完成后自动进入下一步，用户也可以回退修改
3. 计算过程中用户可退出页面，计算完成后会发送邮件/站内通知
4. 结果可随时保存分享，生成分享链接给协作者

## 原型图说明

以上为线框功能原型，具体视觉设计请麟观进行UI美化设计。

核心设计原则：
- 保持左侧步骤导航，引导用户按流程完成仿真
- 主区域用于3D模型显示交互
- 右侧为参数编辑区，布局清晰
- 每一步操作都有明确提示和引导
