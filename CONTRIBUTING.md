# Contributing to CAELab

感谢您对 CAELab 项目的关注！欢迎贡献您的力量。

## 如何贡献

1. **Fork** 本仓库
2. 创建您的特性分支 (`git checkout -b feature/your-feature`)
3. 提交您的更改 (`git commit -m 'Add some feature'`)
4. 推送分支 (`git push origin feature/your-feature`)
5. 开启 **Pull Request**

## 开发环境

请参阅 `README.md` 了解项目设置和运行方式。

## 代码规范

- 使用 ESLint + Prettier 进行代码格式化
- 遵循 TypeScript 最佳实践
- 保持组件简洁和可复用

## 问题反馈

如果您发现 bug 或有建议，请通过 GitHub Issues 报告。
